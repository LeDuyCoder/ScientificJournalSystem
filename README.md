# 📊 JournalTrendingSystem

A powerful scientific journal trend analysis platform built using a Full-stack JavaScript architecture. The system leverages graph data analytics to map complex academic relationships and utilizes a high-performance caching layer to optimize heavy analytical queries under high concurrent traffic.

## 🚀 Key Features
*   **Trend Analysis Dashboard:** Provides interactive charts (Bar charts, Growth trends, and Node-edge graphs) to visualize publishing patterns.
*   **Academic Graph Mapping:** Connects data points between `Journals`, `Articles`, `Authors`, `Keywords`, `Areas`, and `Categories` using a Graph Database.
*   **Smart Query Caching:** Bypasses heavy analytical graph processing by caching repetitive requests to maintain microsecond response times.
*   **Project Workspace:** Allows researchers to create custom workspaces, select domains, and track target keywords over time.
